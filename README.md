# Powershell Lesson 1 

Learn the basics of objects and their use case (scalar and collections)
-	Int
-	String
-	Here Strings
-	Array e.g. @()
-	HashTable e.g. @{}
-	Type accelerators e.g. [System.Collections.ArrayList]
-	CustomObject e.g. [PsCustomObject]@{}
